To get started, you can use the [Vite CLI](https://vitejs.dev/guide/#scaffolding-your-first-vite-project) to initialize the project:

```bash
pnpm create vite@latest my-react-app --template react
cd my-react-app
pnpm install
pnpm run dev
```
